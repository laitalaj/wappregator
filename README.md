<h1 align="center">Wappregator</h1>

<p align="center">
An aggregator for Wappuradios.
Check it out at <a href=https://wappregat.org>https://wappregat.org</a>!
</p>

## Dev quickstart

1. [Get `podman` and `podman-compose`](https://podman.io/)
2. `podman compose --podman-run-args="--rm" up`
3. Check the aggregator at http://localhost:3000/ or the API docs at http://127.0.0.1:8000/docs
4. Check the READMEs under each of the components - `backend`, `pollers`, `common`, `frontend` and `tofu` - for more details.
